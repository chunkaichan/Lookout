# Lookout iOS
> An app designed for elderly safety care. Falls can be serious to the elderly and the rescue time should be as short as possible. While detecting a fall, Lookout immediately sends push notifications and emails to the userâ€™s emergency contacts. <br><br>
> App Store: https://itunes.apple.com/tw/app/lookout/id1170769217?mt=8

## Feature
  * Passwordless login
    * Phone / email registration
  * Home
    * Instant alert
    * Hospitals map
  * Contacts
    * Phone call
    * Location trackings
  * Profile
    * Basic / health information
  * Alert System
    * Fall detection
    * Sending push notification / email to contacts automatically


## Libraries
* Facebook AccountKit
* Firebase
* Crashlytics
* Charts
* GoogleAPIClientForREST/Gmail
* GTMOAuth2
* Mailcore2
* SDWebImage
